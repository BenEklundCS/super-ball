# Super Ball

[![Godot Engine](https://img.shields.io/badge/Godot-4.4%2B-478CBF?style=for-the-badge&logo=godot-engine&logoColor=white)](https://godotengine.org/)
[![C#](https://img.shields.io/badge/C%23-239120?style=for-the-badge&logo=c-sharp&logoColor=white)](https://docs.microsoft.com/en-us/dotnet/csharp/)

**Super Ball** Is a mini 3d demo of racing physics with a rolling ball representing the player, level loading/data persistence, and a single level to race through.

## Controls (Default)
- Move: W (Speed Up), S (Slow Down)
---

## Development
- Engine: Godot 4.4+
- Language: C#
- Scaffold: new-godot-project

### Requirements
- Godot 4.4 or later (Mono build)
- .NET SDK 8.0+ (for C# builds)

### Running
    $ git clone https://github.com/BenEklundCS/super-ball.git
    $ cd super-ball
Open the project in Godot 4.4+ (Mono) and run the main scene.
